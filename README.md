# google_interview_question_database
This is a database for Google SWE Summer Internship Interview Questions - Shanghai &amp; Taiwan

Our team use [Pardus AI](https://pardusai.org/) to get the report

## Description
This dataset contains 122 interview questionscollected from multiple sources specifically for Google SWE Software Engineer Summer Internship positions in Shanghai and Taiwan. The questions are compiled from three primary sources: VerveCopilot's Google LeetCode guide, GeeksforGeeks interview experiences, and Glassdoor interview reports. The data covers a comprehensive range of algorithms and data structures that candidates should prepare for when interviewing with Google in these regions.
